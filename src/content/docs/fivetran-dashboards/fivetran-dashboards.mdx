This is a guide to getting started with New Relic and Fivetran. You will learn how to create a dashboard and setup alert conditions to detect any failed sync jobs.


## Dashboard Quick start [#dashboard-quick-start]
Using the json template below you can create a dashboard to monitor Fivetran jobs in 2 steps.

1. Configure Fivetran external logs to your New Relic account. Please follow instructions here. https://fivetran.com/docs/logs/external-logs/new-relic/setup-guide

## Alert Conditions [#alert-conditions]

### Connection Failure

This alert is triggered in the event of a connection failure, indicating the failure of any Fivetran jobs. Create an alert condition using the parameters below.


| Condition Parameter       | Value
------------ | ------------
| Type | STATIC
| NRQL | SELECT count(*) from Log where event = 'connection_failure'
| valueFunction | SINGLE_VALUE
| priority | CRITICAL
| operator | ABOVE_OR_EQUALS
| threshold | 1
| thresholdDuration | 300
| thresholdOccurrences | AT_LEAST_ONCE


### Task Failure

This alert is triggered in the event of a connection failure, indicating the failure of any Fivetran task.


| Condition Parameter       | Value
------------ | ------------
| Type | STATIC
| NRQL | SELECT count(*) from Log where event = 'sync_end' and data.status ='FAILURE_WITH_TASK'
| valueFunction | SINGLE_VALUE
| priority | CRITICAL
| operator | ABOVE_OR_EQUALS
| threshold | 1
| thresholdDuration | 300
| thresholdOccurrences | AT_LEAST_ONCE
